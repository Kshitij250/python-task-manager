# python-task-manager
A Python CLI task manager that supports task creation, priority assignment, completion tracking, and deletion using core Python data structures.
# Features
> Add new tasks with priority levels (High / Medium / Low)
> View all tasks with status (Pending / Completed)
> Mark tasks as completed
> Remove tasks from the list
> Clean and interactive CLI menu

# Technologies Used
Python
Lists & Dictionaries
Conditional statements and loops
